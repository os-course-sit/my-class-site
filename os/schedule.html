راهنمای سریع برای استفاده

  1. این فایل را ذخیره کن با نام index.html.
  2. یک حساب GitHub جدید بساز (یا از حسابی که می‌خواهی استفاده کنی). برای مخزن `username.github.io` یک repo جدید بساز.
  3. فایل را در ریشه‌ی مخزن قرار بده و Commit کن. سپس در Settings → Pages شاخه را روی main و پوشه / (root) قرار بده.
  4. چند دقیقه صبر کن تا سایت فعال شود. آدرس: https://username.github.io
